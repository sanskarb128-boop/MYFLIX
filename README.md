# 🎬 NetFlimix

NetFlimix is a modern movie discovery web application built with React.js and Vite, powered by the TMDB API. Users can browse popular movies, search movies in real time, view detailed movie information, watch trailers, and save their favorite movies.

## 🚀 Live Demo

🔗 https://netflimix.netlify.app/

## 💻 GitHub Repository

🔗 https://github.com/sanskarb128-boop/MYFLIX

## ✨ Features

* 🔍 Real-time movie search
* 🎞️ Browse popular movies
* ❤️ Add and remove favorites
* 💾 Favorites saved using Local Storage
* 📄 Detailed movie information page
* ⭐ Movie ratings and release dates
* 🎬 Watch movie trailers
* ⚡ Skeleton loading animations
* 📱 Responsive design for desktop and mobile
* 🌐 Deployed on Netlify

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* React Router DOM
* Context API
* CSS3

### API

* TMDB (The Movie Database) API

### Deployment

* Netlify

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/sanskarb128-boop/MYFLIX.git
```

Navigate to the project folder:

```bash
cd MYFLIX/myflix
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
